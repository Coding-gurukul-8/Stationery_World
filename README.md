# Stationery_World

A full-stack stationery e-commerce project (frontend + backend).

## 📌 Workspace structure
- `backend/` — Node.js + Express API with Prisma & PostgreSQL
- `frontend/` — React + Vite UI

## 🚀 Running locally
### Backend
```bash
cd backend
npm install
cp .env.example .env
# update .env with your DB and SMTP credentials
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
